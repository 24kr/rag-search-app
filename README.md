# RAG Search App

A Retrieval-Augmented Generation (RAG) application built with Next.js that allows users to upload documents, search through them using semantic search, and get AI-generated answers based on document content.

## Features

- **Document Upload**: Upload PDF, TXT, and DOCX files
- **Semantic Search**: Search documents using natural language queries
- **AI-Powered Answers**: Get contextual answers generated by Google Gemini AI
- **Vector Embeddings**: Uses Google's embedding-001 model for semantic understanding
- **Document Management**: View and manage uploaded documents
- **PDF Viewer**: Built-in PDF viewer for document preview

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **AI/ML**: Google Generative AI (Gemini)
- **Vector Database**: Pinecone
- **Styling**: Tailwind CSS
- **Language**: TypeScript

## Getting Started

### Prerequisites

- Node.js 18+
- Google AI API key
- Pinecone API key (optional)

### Environment Variables

Create a `.env.local` file:

```env
GOOGLE_API_KEY=your_google_api_key
PINECONE_API_KEY=your_pinecone_api_key
```

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to use the application.

## Usage

1. **Upload Documents**: Click the upload button to add PDF, TXT, or DOCX files
2. **Search**: Enter a natural language query to search across your documents
3. **View Results**: Get AI-generated answers with source citations
4. **Manage Documents**: View and manage uploaded documents in the Documents page

## API Routes

- `POST /api/upload` - Upload and process documents
- `POST /api/search` - Perform RAG search
- `GET /api/documents` - List all documents
